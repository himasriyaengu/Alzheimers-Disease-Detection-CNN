# Alzheimer's Disease Detection using Deep Learning

## Overview

This project presents a deep learning-based framework for Alzheimer's Disease diagnosis using MRI brain scans.

The system focuses on hippocampus-based MRI slice selection and evaluates multiple Convolutional Neural Network architectures including:

- LeNet
- ResNet50
- Extended LeNet

The project utilizes MRI images from the Alzheimer's Disease Neuroimaging Initiative (ADNI) dataset and aims to improve diagnostic accuracy by selecting informative brain regions instead of using full MRI scans.

---

## Features

- MRI image preprocessing
- Hippocampus-based slice selection
- Deep learning classification
- Alzheimer's disease prediction
- Comparative evaluation of CNN architectures
